# JJIF Competition Analysis Dashboard

A Streamlit dashboard for analyzing JJIF (Ju-Jitsu International Federation) competition data, athlete profiles, and tournament brackets.

## Features

- **Athlete Profiles**: View athlete competition history and statistics
- **Competition Results**: Browse and filter match results
- **Head-to-Head Analysis**: Compare athletes' performance records
- **Bracket Visualization**: View tournament bracket progressions
- **Rankings**: Track world and continental rankings

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run dashboard
streamlit run dashboard.py
```

## Data

This dashboard uses a private data submodule. To clone with data:

```bash
git clone --recurse-submodules https://github.com/LukeJGallagher/jiujitsu-dashboard.git
```

## Weight Categories (JJIF)

**Male**: 56kg, 62kg, 69kg, 77kg, 85kg, 94kg, +94kg
**Female**: 48kg, 52kg, 57kg, 63kg, 70kg, +70kg
